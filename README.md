# USG Console – projekt scalony (probna + asysta)

## Jak uruchomić
1. Rozpakuj ZIP.
2. Otwórz plik `index.html` w przeglądarce (Chrome/Edge/Firefox).

## Co zostało zrobione
- `index.html` to **probna.html** jako baza (wygląd + logika symulatora bez zmian).
- Dodana jest tylko **warstwa edukacyjna** (skopiowana/zaadaptowana z asysta.html):
  - dopisuje sekcję „EDUKACJA (asysta)” w dymku asystenta,
  - dopisuje 4 dodatkowe statystyki w „Metrykach technicznych” (axial/lateral/penetration/SNR),
  - uzupełnia stopkę asystenta o te statystyki.
- Nie ma zewnętrznych zależności.

## Pliki
- `index.html` – gotowa wersja produkcyjna (otwierać).
- `probna.html`, `asysta.html` – oryginały do wglądu.
- `logo.png`, `piotr.png` – grafiki.
